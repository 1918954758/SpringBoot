```text
to map 404 to a static HTML file, your directory structure would be as follows:
src/
 +- main/
     +- java/

     +- resources/

             +- error/
             |   +- 404.html
             +- <other public assets>
```
or
```text
To map all 5xx errors by using a FreeMarker template, your directory structure would be as follows:
src/
 +- main/

     |   + <source code>
     +- resources/

             +- error/
             |   +- 5xx.ftlh
```
